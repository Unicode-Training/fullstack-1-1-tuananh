# Tailwind

## Concept

- Theo concept utility-first
- Tách nhỏ các thuộc tính và giá trị thành các class

## Công thức class

```html
tiento-giatri (giatri do tailwind build sẵn)
```

```html
tiento-[giatri] (giatri tự đưa vào)
```

```html
pseudo:tiento-giatri
```

```html
pseudo:tiento-[giatri]
```

```html
breakpoint:tiento-giatri
```

```html
breakpoint:tiento-[giatri]
```

Giao diện sáng tối

- Concept: Dựa vào class dark ở thẻ html
- Dùng cú pháp: dark:tiento-giatri hoặc dark:tiento-[giatri]
